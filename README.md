# LedGix ERP

**LedGix ERP** is a modern, cloud-based ERP platform designed for SMEs, trading companies, and multi-branch organizations. It provides a robust suite of tools for accounting, sales, inventory, and more, built on a foundation of accuracy, scalability, and ease of use.

## 🚀 Product Overview
LedGix ERP leverages Flutter for a seamless cross-platform experience (Web, Android, iOS) and Firebase for a high-performance, real-time backend. Our goal is to provide enterprise-grade features with a focus on simple, intuitive design and strict accounting principles.

## 🛠 Tech Stack
- **Frontend:** Flutter (Web & Mobile)
- **Backend:** Firebase (Auth, Firestore, Cloud Functions, Storage)
- **Design:** Pure Black Theme, Responsive UI
- **Architecture:** Feature-based Clean Architecture

## ⚖️ Development Principles
- **Accuracy First:** Double-entry accounting is the heart of the system.
- **Auditability:** Every change is logged; posted documents are immutable.
- **Performance:** Optimized for speed and low-bandwidth responsiveness.
- **Security:** Strict data isolation and role-based access control.

## 🏁 Getting Started
1. **Prerequisites:** Install Flutter and Firebase CLI.
2. **Setup:** Run `flutter pub get`.
3. **Run:** Use `flutter run -d chrome` for web or select a mobile device.
