# LedGix ERP Documentation

Welcome to the central documentation repository for **LedGix ERP**. This ecosystem serves as the single source of truth for all stakeholders, including developers, product owners, accountants, and designers.

## Product Overview
LedGix ERP is a modern, cloud-based ERP platform designed for SMEs, trading, service, manufacturing, and construction companies. Built with Flutter and Firebase, it prioritizes accounting accuracy, scalability, and auditability.
