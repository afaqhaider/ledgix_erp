# 17 Audit & Compliance

## Audit Trail
- **Document History:** Every change to an invoice, bill, or journal entry is tracked.
- **Access Logs:** Tracking user logins and sensitive data exports.
- **Immutable Ledger:** Once a transaction is "Posted", it cannot be edited or deleted. Corrective actions must be via "Reversal" or "Adjustment" entries.

## Financial Controls
- **Segregation of Duties:** Ensure one user cannot create and approve their own transaction (configurable per company).
- **Approval Limits:** Expenditure thresholds requiring higher-level approval.
- **Bank Reconciliation:** Tools to match system transactions with bank statements.

## Compliance Standards
- **VAT/Tax:** Configurable tax engines to support multiple jurisdictions.
- **IFRS Alignment:** Reporting formats following International Financial Reporting Standards.
- **Data Privacy:** GDPR/CCPA readiness through data deletion tools and export capability.

## Document Revisions
- Support for versioning on major documents (Quotes, Contracts).
- Ability to view "Previous Version" side-by-side with current.
